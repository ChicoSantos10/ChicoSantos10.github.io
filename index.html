AI

I was responsible for 3 main systems:

Enemy behavior

Created a behavior tree with a custom tool that allowed easy creation and debugging of enemy behavior

Enemy pathfinding

Implemented an A* algorithm in conjunction with a map node generator.

Map generation

I implemented a map generator using a multistep process. First we define the center of each biome. This allows for rule one to be kept: the spawn biome should be in the center and then the other should form a circle around it. Then a version of a flood algorithm starting on each biome center fills the map with the different biomes. By using some randomness here I guarantee that no biome would be the same while also create some "holes" that do not allow the player to traverse. Then it cleans the map by making sure that all nodes of a certain biome are connected. It then adds colliders so that the player can only walk through the walkable biomes. It then creates the map texture by using a compute shader using the biomes around a particular node to figure out what textures to blend. To finish the map it uses a Poisson disc sampler to place assets and interactable objects.
